Política de privacidad

Texto orientativo. Revísalo antes de publicar definitivamente y evita recoger datos clínicos o información innecesaria.

Datos tratados

Se recogen los datos introducidos en el formulario: nombre, email, fecha y hora preferidas, nota opcional, código de solicitud y aceptación de condiciones.

Finalidad

Gestionar solicitudes, coordinar encuentros online, confirmar disponibilidad, tramitar pagos y responder comunicaciones relacionadas con el servicio.

Base jurídica

La base jurídica es la solicitud de medidas precontractuales, la ejecución del servicio solicitado y el consentimiento prestado al enviar el formulario.

Conservación

Los datos se conservarán durante el tiempo necesario para gestionar la solicitud y cumplir obligaciones administrativas, fiscales o legales aplicables.

Destinatarios

Podrán intervenir proveedores técnicos necesarios para el funcionamiento de la web, el formulario y el pago, como Formspree, Stripe, GitHub Pages o servicios equivalentes.

Derechos

Puedes solicitar acceso, rectificación, supresión, oposición, limitación o portabilidad de tus datos escribiendo a espacioungulo@protonmail.com. También puedes reclamar ante la Agencia Española de Protección de Datos.

No introducción de datos clínicos

El formulario no está destinado a recoger datos médicos, diagnósticos, información clínica o situaciones de urgencia. La nota opcional debe ser general y breve.
